# Idol Event Landing Page Skill

Generate event websites, poster announcements, timetable cards, ticket conversion pages, and fan quick guides.

## Required Inputs

- Event name, date, venue, city
- Performing groups and members
- Ticket tiers, merch rules, timetable
- Source URL or calendar entry

## Output Contract

- Event landing HTML
- Timetable block
- Ticket CTA copy
- Fan quick guide section

## Checklist

- Date, venue, and ticket action must be visible above the fold.
- Separate official information from inferred platform notes.
- Include mobile-first timetable readability.
- Use only licensed or authorized poster, venue, and performer assets.
- Do not imply official endorsement unless the source explicitly supports it.
- Preserve prompt and timestamp in the artifact metadata.
