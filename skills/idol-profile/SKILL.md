# Idol Profile Page Skill

Generate member profile pages, social cards, birthday campaign pages, cheki merch cards, and otome-style character pages.

## Required Inputs

- Member name, nickname, color, symbol, tags
- Group or solo status
- Social links and ranking signals when available
- Target audience: fans, first-time discovery, sponsors, or press

## Output Contract

- Single HTML artifact
- Markdown summary
- Suggested social card copy
- Source notes for ranking and profile data

## Checklist

- Keep the member identity visible in the first viewport.
- Use the member color as a signal, not as the whole palette.
- Include fan-facing CTA and source-aware data notes.
- Label generated real-idol visuals as `AI generated`.
- Do not impersonate the idol's private speech or endorsement.
- Do not create adultized, maliciously humiliating, or unauthorized deepfake content.
